Scenario Guide

GREEN: Successful Backup
Full backup pipeline executes: GREEN status + changes detected + RTB creates snapshot + pCloud uploads incremental data with deduplication (35 minutes total)
YELLOW: Warning Status
Elevated entropy or ClamAV alert detected. If SAFETY_GATE_STRICT=1 (recommended), backup is blocked. Otherwise runs with caution (8 seconds)
RED: Ransomware Alert
Ransomware/virus signature detected. Backup immediately aborted. Critical security incident logged to database (8 seconds)

GREEN status but no file modifications since last snapshot. Backup skipped to save CPU, disk I/O, and bandwidth (9 seconds)
